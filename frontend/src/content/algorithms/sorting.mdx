# Algoritmos de ordenamiento

Los algoritmos de ordenamiento son fundamentales en ciencias de la computación.

## Bubble Sort — O(n²)

<CodeRunner
  language="python"
  title="Bubble Sort"
  initialCode={`
def bubble_sort(arr):
    n = len(arr)
    arr = arr.copy()
    for i in range(n):
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr

datos = [64, 34, 25, 12, 22, 11, 90]
print("Original:", datos)
print("Ordenado:", bubble_sort(datos))
  `}

## Merge Sort — O(n log n)

<CodeRunner
  language="python"
  title="Merge Sort"
  initialCode={`
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    izq = merge_sort(arr[:mid])
    der = merge_sort(arr[mid:])
    return merge(izq, der)

def merge(izq, der):
    resultado = []
    i = j = 0
    while i < len(izq) and j < len(der):
        if izq[i] <= der[j]:
            resultado.append(izq[i]); i += 1
        else:
            resultado.append(der[j]); j += 1
    return resultado + izq[i:] + der[j:]

datos = [38, 27, 43, 3, 9, 82, 10]
print("Original:", datos)
print("Ordenado:", merge_sort(datos))
  `}
