# Cadenas de Texto (Strings)

Las cadenas de texto o **strings** son uno de los tipos de datos más utilizados en Python. Representan secuencias de caracteres y Python ofrece una cantidad enorme de herramientas para trabajar con ellas.

## Introducción a las cadenas

Un string se crea encerrando texto entre comillas simples `'...'`, dobles `"..."` o triples `'''...'''` / `"""..."""`.

<CodeRunner language="python" title="Crear strings" initialCode={`# Comillas simples o dobles, ambas válidas
nombre = 'Ana'
apellido = "García"

# Útil cuando el texto contiene comillas
frase1 = "No puedo creer que diga 'hola' así"
frase2 = 'Él dijo: "Buenos días"'

print(frase1)
print(frase2)

# Strings multilínea con triple comilla
texto = """Esta es
una cadena
de varias líneas"""
print(texto)`} />

### Caracteres especiales

<CodeRunner language="python" title="Caracteres especiales" initialCode={`# Salto de línea
print("Línea 1\nLínea 2")

# Tabulación
print("Nombre:\tAna")

# Barra invertida literal
print("C:\\Users\\Ana")

# Raw strings: ignoran los caracteres especiales
ruta = r"C:\Users\Ana\Documentos"
print(ruta)`} />

## Operaciones básicas con cadenas

### Concatenación y repetición

Los strings son **inmutables** — no se pueden modificar directamente. Operaciones como la concatenación siempre generan un nuevo string, sin modificar los originales.

<CodeRunner language="python" title="Concatenación y repetición" initialCode={`nombre = "Ana"
apellido = "García"

# Concatenar con +
nombre_completo = nombre + " " + apellido
print(nombre_completo)

# Concatenación genera un nuevo string, los originales no cambian
s = "Hola"
s2 = s + " Mundo"
print(s)   # Hola   ← no ha cambiado
print(s2)  # Hola Mundo

# Repetir con *
separador = "-" * 20
print(separador)

# Concatenar con variables numéricas requiere conversión
edad = 25
print("Tengo " + str(edad) + " años")`} />

### F-strings (la forma moderna)

Las f-strings son la forma recomendada de insertar variables dentro de strings en **Python 3.6+**.

<CodeRunner language="python" title="F-strings" initialCode={`nombre = "Carlos"
edad = 30
altura = 1.75

# Sintaxis: f"hola nombre" con llaves alrededor de nombre
print(f"Me llamo {nombre} y tengo {edad} años")

# Puedes incluir expresiones directamente
print(f"El doble de mi edad es {edad * 2}")
print(f"Altura en cm: {altura * 100:.0f}")

# Formateo de números
pi = 3.14159265
print(f"Pi con 2 decimales: {pi:.2f}")
print(f"Número con separador: {1000000:,}")`} />

## Longitud de una cadena

La función `len()` devuelve el número de caracteres de un string, incluyendo espacios y caracteres especiales.

<CodeRunner language="python" title="len()" initialCode={`texto = "Hola, mundo"
print(len(texto))         # 11

# Los espacios también cuentan
print(len("Ana García"))  # 10

# String vacío
print(len(""))            # 0

# Uso práctico: validar longitud
contrasena = "abc123"
if len(contrasena) < 8:
    print("La contraseña es demasiado corta")
else:
    print("Contraseña válida")`} />

## Indexación

Los strings son secuencias, por lo que puedes acceder a cada carácter por su **índice**. En Python los índices empiezan en `0`. Los índices negativos cuentan desde el final — `-1` apunta siempre al último carácter, `-2` al penúltimo, y así sucesivamente.

<CodeRunner language="python" title="Indexación" initialCode={`texto = "Python"
#         P  y  t  h  o  n
# índice: 0  1  2  3  4  5
#        -6 -5 -4 -3 -2 -1

# Índices positivos (de izquierda a derecha)
print(texto[0])   # P
print(texto[1])   # y
print(texto[5])   # n

# Índices negativos: -1 es siempre el último carácter
print(texto[-1])  # n (último)
print(texto[-2])  # o (penúltimo)
print(texto[-6])  # P (primero, equivale a texto[0])

# Los strings son inmutables: no puedes modificar un carácter
try:
    texto[0] = "p"
except TypeError as e:
    print("Error:", e)`} />

## Subcadenas (slicing)

El slicing permite extraer una porción de un string con la sintaxis `[inicio:fin:paso]`:
- `inicio` — índice incluido (por defecto: principio del string)
- `fin` — índice excluido (por defecto: final del string)
- `paso` — salto entre caracteres (por defecto: 1)

<CodeRunner language="python" title="Slicing básico" initialCode={`texto = "Python es genial"

# [inicio:fin] — inicio incluido, fin excluido
print(texto[0:6])    # Python
print(texto[7:9])    # es
print(texto[10:])    # genial (hasta el final)
print(texto[:6])     # Python (desde el inicio)
print(texto[:])      # copia completa

# Índices negativos en slicing
print(texto[-6:])    # genial
print(texto[:-10])   # Python`} />

<CodeRunner language="python" title="Slicing con paso" initialCode={`texto = "Python"

# Paso: cuántos caracteres saltar
print(texto[::2])    # Pto  (cada 2 caracteres)
print(texto[1::2])   # yhn  (desde índice 1, cada 2)

# Paso negativo: recorre el string al revés
print(texto[::-1])   # nohtyP

# Invertir cualquier string es tan simple como:
frase = "Hola mundo"
print(frase[::-1])   # odnum aloH`} />

## Búsqueda en cadenas

Python ofrece varias formas de buscar texto dentro de un string.

<CodeRunner language="python" title="Búsqueda en strings" initialCode={`texto = "Python es un lenguaje de programación"

# in / not in: comprueba si existe
print("Python" in texto)        # True
print("Java" in texto)          # False
print("Java" not in texto)      # True

# find(): devuelve el índice o -1 si no encuentra
print(texto.find("es"))         # 7
print(texto.find("Java"))       # -1

# index(): igual que find() pero lanza error si no encuentra
print(texto.index("es"))        # 7

# count(): cuenta las ocurrencias
texto2 = "banana"
print(texto2.count("a"))        # 3
print(texto2.count("an"))       # 2

# startswith() y endswith() aceptan una tupla de opciones
print(texto.startswith("Python"))              # True
print(texto.startswith(("Python", "Java")))    # True
print(texto.endswith("ción"))                  # True
print(texto.endswith(("ción", "mente")))       # True`} />

## Métodos de cadenas

Los strings tienen decenas de métodos incorporados. Aquí están los más útiles.

### Métodos básicos

<CodeRunner language="python" title="Métodos de transformación" initialCode={`texto = "  Hola, Mundo  "

# Mayúsculas y minúsculas
print(texto.upper())        #   HOLA, MUNDO
print(texto.lower())        #   hola, mundo
print(texto.capitalize())   #   Hola, mundo
print(texto.title())        #   Hola, Mundo
print(texto.swapcase())     #   hOLA, mUNDO

# Eliminar espacios
print(texto.strip())        # "Hola, Mundo"
print(texto.lstrip())       # "Hola, Mundo  "
print(texto.rstrip())       # "  Hola, Mundo"

# Reemplazar
print("banana".replace("a", "o"))    # bonono
print("a,b,c".replace(",", " - "))   # a - b - c`} />

<CodeRunner language="python" title="Métodos de división y unión" initialCode={`# split(): divide un string en una lista
frase = "Python es muy potente"
palabras = frase.split()
print(palabras)           # ['Python', 'es', 'muy', 'potente']

csv = "Ana,Carlos,María,Juan"
nombres = csv.split(",")
print(nombres)            # ['Ana', 'Carlos', 'María', 'Juan']

# join(): une una lista en un string
print(" ".join(palabras))     # Python es muy potente
print(", ".join(nombres))     # Ana, Carlos, María, Juan
print("-".join("Python"))     # P-y-t-h-o-n

# Caso práctico: limpiar y separar entradas del usuario
entrada = "  Ana, Carlos , María  "
nombres_limpios = [n.strip() for n in entrada.split(",")]
print(nombres_limpios)    # ['Ana', 'Carlos', 'María']

# splitlines(): divide por saltos de línea
multilinea = "línea 1\nlínea 2\nlínea 3"
print(multilinea.splitlines())`} />

<CodeRunner language="python" title="Métodos de validación" initialCode={`print("123".isdigit())       # True
print("abc".isalpha())       # True
print("abc123".isalnum())    # True
print("   ".isspace())       # True
print("Hola".islower())      # False
print("HOLA".isupper())      # True

# Uso práctico: validar entrada
entrada = "12345"
if entrada.isdigit():
    numero = int(entrada)
    print(f"Número válido: {numero}")
else:
    print("No es un número válido")`} />

### Método `format()`

Antes de las f-strings, `format()` era la forma estándar de insertar valores en strings. Funciona en Python 2.7 y 3.x, por lo que es muy común encontrarlo en código existente.

<CodeRunner language="python" title="Método format()" initialCode={`# Sintaxis básica con {}
print("Hola, {}!".format("Ana"))
print("Tengo {} años y mido {} m".format(25, 1.75))

# Con índices
print("{0} y {1}".format("Pan", "Agua"))
print("{1} y {0}".format("Pan", "Agua"))  # orden invertido

# Con nombres
print("{nombre} tiene {edad} años".format(nombre="Carlos", edad=30))

# Formateo de números
print("Pi: {:.2f}".format(3.14159))
print("Porcentaje: {:.1%}".format(0.856))
print("Número: {:,}".format(1000000))`} />

<CodeRunner language="python" title="format() vs f-strings" initialCode={`nombre = "Ana"
edad = 25

# format() — compatible con Python 2.7+
print("Me llamo {} y tengo {} años".format(nombre, edad))

# f-string — recomendado en Python 3.6+
print(f"Me llamo {nombre} y tengo {edad} años")

# Con diccionarios
datos = {"nombre": "Carlos", "ciudad": "Madrid"}
print("Nombre: {nombre}, Ciudad: {ciudad}".format(**datos))
print(f"Nombre: {datos['nombre']}, Ciudad: {datos['ciudad']}")`} />

> Usa **f-strings** en código nuevo — son más legibles y ligeramente más rápidas. Conoce `format()` para entender código existente.
