# 3.9 Documentación

A medida que los programas crecen, recordar qué hace cada función se vuelve difícil. Python ofrece un mecanismo integrado para documentar funciones directamente en el código: los **docstrings**. Documentar bien es tan importante como escribir código que funcione.

---

## Docstrings

Un **docstring** es una cadena de texto que se coloca como **primera línea** del cuerpo de una función. Python la almacena automáticamente como documentación accesible:

<CodeRunner language="python" title="Docstring básico" initialCode={`def saludar(nombre):
    """Imprime un saludo personalizado."""
    print(f"Hola, {nombre}")

saludar("Ana")

# Python almacena el docstring en el atributo __doc__
print(saludar.__doc__)`} />

Para funciones más complejas, se usa un docstring **multilínea** que describe qué hace la función, qué parámetros recibe y qué devuelve:

<CodeRunner language="python" title="Docstring multilínea" initialCode={`def calcular_iva(precio, tasa=21):
    """
    Calcula el precio final con IVA incluido.
    
    Parámetros:
        precio: precio base del producto (número positivo)
        tasa: porcentaje de IVA a aplicar (por defecto 21)
    
    Retorna:
        El precio final redondeado a 2 decimales.
    """
    return round(precio * (1 + tasa / 100), 2)

print(calcular_iva(100))
print(calcular_iva(100, 10))`} />

> El docstring siempre usa **comillas triples** (`"""..."""`), incluso si es una sola línea. Debe ser la **primera instrucción** del cuerpo de la función.

## Uso de help()

La función integrada `help()` muestra el docstring de cualquier función de forma formateada. Funciona tanto con funciones propias como con funciones integradas de Python:

<CodeRunner language="python" title="help() con funciones propias" initialCode={`def precio_final(precio, descuento=0, iva=21):
    """
    Calcula el precio final aplicando descuento e IVA.
    
    Parámetros:
        precio: precio base del producto
        descuento: porcentaje de descuento (por defecto 0)
        iva: porcentaje de IVA (por defecto 21)
    
    Retorna:
        Precio final redondeado a 2 decimales.
    """
    base = precio * (1 - descuento / 100)
    return round(base * (1 + iva / 100), 2)

# help() muestra la documentación completa
help(precio_final)`} />

<CodeRunner language="python" title="help() con funciones integradas" initialCode={`# help() funciona con cualquier función de Python
help(abs)
print("---")
help(round)
print("---")
help(len)`} />

<CodeRunner language="python" title="Función sin docstring" initialCode={`def sumar(a, b):
    return a + b

# Sin docstring, help() muestra solo la firma
help(sumar)

# __doc__ es None
print(sumar.__doc__)`} />

> Escribe un docstring en toda función que no sea trivial. Tu yo del futuro (y tus compañeros) lo agradecerán cuando necesiten entender qué hace una función sin leer todo su código.

## Ejemplo: area_circulo()

Veamos un ejemplo completo de una función bien documentada, desde la definición hasta su uso con `help()`:

<CodeRunner language="python" title="Función area_circulo() documentada" initialCode={`import math

def area_circulo(radio):
    """
    Calcula el área de un círculo dado su radio.
    
    Parámetros:
        radio: radio del círculo (número positivo)
    
    Retorna:
        Área del círculo redondeada a 4 decimales.
        Devuelve None si el radio no es válido.
    """
    if radio < 0:
        print("Error: el radio no puede ser negativo")
        return None
    return round(math.pi * radio ** 2, 4)

# Ver la documentación
help(area_circulo)

print("---")

# Usar la función
print(area_circulo(5))      # 78.5398
print(area_circulo(1))      # 3.1416
print(area_circulo(0))      # 0
print(area_circulo(-3))     # None`} />

<CodeRunner language="python" title="Docstrings en un programa completo" initialCode={`import math

def area_circulo(radio):
    """Calcula el área de un círculo."""
    return round(math.pi * radio ** 2, 4)

def perimetro_circulo(radio):
    """Calcula el perímetro de un círculo."""
    return round(2 * math.pi * radio, 4)

def describir_circulo(radio):
    """
    Muestra las propiedades principales de un círculo.
    
    Parámetros:
        radio: radio del círculo (número positivo)
    """
    print(f"Círculo de radio {radio}:")
    print(f"  Diámetro:  {radio * 2}")
    print(f"  Área:      {area_circulo(radio)}")
    print(f"  Perímetro: {perimetro_circulo(radio)}")

# Cada función tiene su propia documentación
describir_circulo(5)
print()
describir_circulo(10)`} />
