# 3.8 Funciones matemáticas (módulo math)

Python incluye el módulo `math` en su biblioteca estándar, que proporciona funciones matemáticas avanzadas y constantes de uso frecuente. A diferencia de las funciones integradas como `abs()` o `round()`, las funciones de `math` requieren **importar** el módulo antes de usarlas.

---

## Importación

Para usar cualquier función o constante de `math`, primero hay que importar el módulo:

<CodeRunner language="python" title="Formas de importar math" initialCode={`# Forma 1: importar el módulo completo
import math
print(math.sqrt(25))   # 5.0

# Forma 2: importar funciones específicas
from math import sqrt, pi
print(sqrt(25))        # 5.0
print(pi)              # 3.141592653589793

# Forma 3: importar con alias (menos común)
import math as m
print(m.sqrt(25))      # 5.0`} />

> La forma recomendada es `import math` y usar `math.funcion()`. Deja claro de dónde viene cada función y evita conflictos de nombres.

---

## Constantes (pi, e)

El módulo `math` incluye las constantes matemáticas más importantes con alta precisión:

<CodeRunner language="python" title="Constantes matemáticas" initialCode={`import math

# pi: razón entre circunferencia y diámetro
print(f"pi = {math.pi}")            # 3.141592653589793

# e (número de Euler): base del logaritmo natural
print(f"e = {math.e}")             # 2.718281828459045

# tau: 2pi, el ángulo completo en radianes
print(f"tau = {math.tau}")           # 6.283185307179586

# Infinito
print(f"inf = {math.inf}")        # inf

# Ejemplo: área y perímetro de un círculo
radio = 5
area = math.pi * radio ** 2
perimetro = 2 * math.pi * radio
print(f"Radio: {radio}")
print(f"Área: {round(area, 2)}")
print(f"Perímetro: {round(perimetro, 2)}")`} />

## Raíces y potencias

<CodeRunner language="python" title="Raíces y potencias" initialCode={`import math

# Raíz cuadrada
print(math.sqrt(144))      # 12.0
print(math.sqrt(2))        # 1.4142135623730951

# Para otras raíces, se usa exponente fraccionario
print(27 ** (1/3))         # 3.0  (raíz cúbica)

# Potencia (equivalente a ** pero devuelve float)
print(math.pow(2, 10))     # 1024.0

# Diferencia con pow() integrado:
# math.pow() siempre devuelve float
# pow() integrado preserva el tipo
print(type(math.pow(2, 3)))   # <class 'float'>
print(type(pow(2, 3)))        # <class 'int'>`} />

## Redondeo

El módulo `math` ofrece funciones de redondeo más específicas que `round()`:

<CodeRunner language="python" title="Funciones de redondeo" initialCode={`import math

valor = 3.7

# floor(): redondea hacia abajo (al entero menor o igual)
print(math.floor(3.7))     # 3
print(math.floor(-3.7))    # -4

# ceil(): redondea hacia arriba (al entero mayor o igual)
print(math.ceil(3.2))      # 4
print(math.ceil(-3.2))     # -3

# trunc(): trunca la parte decimal (hacia cero)
print(math.trunc(3.9))     # 3
print(math.trunc(-3.9))    # -3`} />

## Logaritmos

<CodeRunner language="python" title="Logaritmos" initialCode={`import math

# log(): logaritmo natural (base e)
print(math.log(math.e))      # 1.0
print(math.log(1))           # 0.0

# log() con base personalizada
print(math.log(100, 10))     # 2.0  (log base 10 de 100)
print(math.log(8, 2))        # 3.0  (log base 2 de 8)

# log10(): logaritmo base 10 (más preciso que log(x, 10))
print(math.log10(1000))      # 3.0

# log2(): logaritmo base 2 (más preciso que log(x, 2))
print(math.log2(1024))       # 10.0`} />

<CodeRunner language="python" title="Ejemplo: escala de magnitud" initialCode={`import math

# Los logaritmos se usan para medir magnitudes en escala
# Ejemplo: ¿cuántos dígitos tiene un número?
def num_digitos(n):
    if n == 0:
        return 1
    return math.floor(math.log10(abs(n))) + 1

print(num_digitos(0))         # 1
print(num_digitos(42))        # 2
print(num_digitos(1000))      # 4
print(num_digitos(9999999))   # 7`} />

## Trigonometría

Las funciones trigonométricas de `math` trabajan en **radianes**, no en grados:

<CodeRunner language="python" title="Funciones trigonométricas" initialCode={`import math

# Conversión grados ↔ radianes
print(math.radians(180))     # 3.141592... (π)
print(math.degrees(math.pi)) # 180.0

# Seno, coseno, tangente
angulo = math.radians(30)   # 30°
print(f"sin(30°) = {round(math.sin(angulo), 4)}")    # 0.5
print(f"cos(30°) = {round(math.cos(angulo), 4)}")    # 0.866
print(f"tan(30°) = {round(math.tan(angulo), 4)}")    # 0.5774

# Valores conocidos
print(f"sin(90°) = {math.sin(math.radians(90))}")    # 1.0
print(f"cos(0°)  = {math.cos(math.radians(0))}")     # 1.0`} />

<CodeRunner language="python" title="Ejemplo: distancia entre dos puntos" initialCode={`import math

def distancia(x1, y1, x2, y2):
    """Calcula la distancia euclidiana entre dos puntos."""
    return math.sqrt((x2 - x1) ** 2 + (y2 - y1) ** 2)

# Distancia entre (0, 0) y (3, 4)
d = distancia(0, 0, 3, 4)
print(f"Distancia: {d}")   # 5.0

# Distancia entre dos ciudades (coordenadas ficticias en km)
d2 = distancia(10, 20, 35, 50)
print(f"Distancia: {round(d2, 2)} km")`} />
