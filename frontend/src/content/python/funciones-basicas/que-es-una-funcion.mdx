# 3.1 ¿Qué es una función?

A medida que los programas crecen, el código tiende a repetirse y a volverse difícil de leer. Las **funciones** son la herramienta fundamental de Python para organizar, reutilizar y estructurar el código en bloques con nombre y propósito claro.

---

## Definición

Una **función** es un bloque de código con nombre que realiza una tarea específica y puede ejecutarse tantas veces como sea necesario simplemente llamándola por su nombre.

Ya hemos usado funciones desde el principio del curso: `print()`, `len()`, `type()`, `abs()`, `round()` son todas funciones. La diferencia es que esas funciones las proporciona Python. En este capítulo aprenderemos a crear las nuestras.

---

## Reutilización de código

Sin funciones, cualquier comportamiento que se necesite en varios puntos del programa debe copiarse. Esto crea problemas inmediatos: si hay un error o se necesita un cambio, hay que encontrar y modificar todas las copias.

Con una función, el comportamiento se define una sola vez y se usa tantas veces como sea necesario. Un cambio en la función afecta automáticamente a todos los lugares donde se usa:

print(f"Producto 1: {precio_con_iva(100)}€")

## Modularidad

Las funciones permiten dividir un programa complejo en piezas más pequeñas, cada una con una responsabilidad clara. Este principio se llama **modularidad**: en lugar de un bloque largo de código difícil de entender, el programa se construye como una combinación de funciones bien nombradas.

Un programa modular es más fácil de:

- **Leer**: cada función tiene un nombre que describe lo que hace
- **Depurar**: se puede probar cada función por separado
- **Mantener**: un cambio en una función no afecta a las demás
- **Reutilizar**: una función bien diseñada puede usarse en otros programas

<CodeRunner language="python" title="Programa modular" initialCode={`def calcular_descuento(precio, porcentaje):
    return precio * (porcentaje / 100)

def calcular_iva(precio):
    return precio * 0.21

def precio_final(precio, descuento=0):
    precio_con_descuento = precio - calcular_descuento(precio, descuento)
    return round(precio_con_descuento + calcular_iva(precio_con_descuento), 2)

# El programa principal es fácil de leer
print(precio_final(100))          # sin descuento
print(precio_final(100, 10))      # con 10% de descuento
print(precio_final(250, 15))      # con 15% de descuento`} />

## Las funciones como valores

En Python, una función es un objeto como cualquier otro. Puede inspeccionarse, asignarse a una variable o pasarse como argumento:

<CodeRunner language="python" title="Las funciones son objetos" initialCode={`def saludar():
    print("Hola")

# La función es un objeto de tipo function
print(type(saludar))    # <class 'function'>
print(saludar)          # <function saludar at 0x...>

# Sin paréntesis: referencia al objeto función (no la ejecuta)
# Con paréntesis: llamada que ejecuta el cuerpo
saludar()               # Hola`} />

> Sin los paréntesis, `saludar` es solo una referencia al objeto función. Con los paréntesis, `saludar()` ejecuta el cuerpo. Este es uno de los errores más frecuentes al empezar a trabajar con funciones.
