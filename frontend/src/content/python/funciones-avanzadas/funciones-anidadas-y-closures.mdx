# 5.2 Funciones anidadas y closures

Python permite definir funciones **dentro** de otras funciones. Estas funciones internas pueden acceder a las variables de la función que las contiene, creando un mecanismo poderoso llamado **closure**.

---

## Funciones dentro de funciones

Una función definida dentro de otra solo existe dentro del ámbito de la función contenedora. No es accesible desde fuera:

## Captura de variables

Lo más interesante de las funciones anidadas es que **capturan** las variables de la función contenedora. Esto se llama **closure**: la función interna "recuerda" el entorno donde fue creada, incluso después de que la función contenedora haya terminado:

<CodeRunner language="python" title="Closure: captura de variables" initialCode={`def crear_contador(inicio=0):
    """Crea una función que cuenta desde 'inicio'."""
    cuenta = [inicio]   # lista para poder modificar desde el closure
    
    def contar():
        cuenta[0] += 1
        return cuenta[0]
    
    return contar

# La función 'contar' recuerda su propia 'cuenta'
contador_a = crear_contador(0)
contador_b = crear_contador(100)

print(contador_a())   # 1
print(contador_a())   # 2
print(contador_a())   # 3

print(contador_b())   # 101
print(contador_b())   # 102`} />

<CodeRunner language="python" title="Closure con nonlocal" initialCode={`def crear_acumulador(inicio=0):
    """Crea una función que acumula valores."""
    total = inicio
    
    def agregar(valor):
        nonlocal total   # permite modificar 'total' del closure
        total += valor
        return total
    
    return agregar

cuenta = crear_acumulador()
print(cuenta(10))    # 10
print(cuenta(25))    # 35
print(cuenta(5))     # 40

# Cada acumulador es independiente
otra_cuenta = crear_acumulador(100)
print(otra_cuenta(50))   # 150`} />

> La palabra clave `nonlocal` permite que una función anidada **modifique** una variable de la función contenedora. Sin `nonlocal`, solo puede leerla.

## Generadores de funciones

Los closures permiten crear **fábricas de funciones**: funciones que generan funciones especializadas según los parámetros recibidos:

<CodeRunner language="python" title="Fábrica de funciones matemáticas" initialCode={`def crear_potencia(exponente):
    """Crea una función que eleva a un exponente fijo."""
    def potencia(base):
        return base ** exponente
    return potencia

cuadrado = crear_potencia(2)
cubo = crear_potencia(3)
raiz_cuadrada = crear_potencia(0.5)

print(cuadrado(5))          # 25
print(cubo(3))              # 27
print(raiz_cuadrada(16))    # 4.0

# Se pueden usar en listas
numeros = [2, 3, 4, 5]
print(list(map(cuadrado, numeros)))   # [4, 9, 16, 25]
print(list(map(cubo, numeros)))       # [8, 27, 64, 125]`} />
